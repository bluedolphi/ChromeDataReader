Chrome数据读取工具 - 单文件版本
===================================

版本：1.0 (单文件版)
发布日期：2025-09-16
文件大小：约 155MB

## 文件说明

ChromeDataReader-SingleFile.exe - 完整的独立可执行文件
- 包含所有依赖库和运行时
- 无需安装任何额外软件
- 真正的"绿色软件"，单文件运行

## 系统要求

- Windows 10/11 操作系统 (x64)
- 已安装 Chrome 浏览器
- 建议以管理员权限运行

## 使用方法

1. 双击 ChromeDataReader-SingleFile.exe 启动程序
2. 程序会自动检测系统中的 Chrome 安装
3. 使用界面中的按钮读取不同类型的数据：
   - 读取 Cookie 数据
   - 读取 Session 数据
   - 读取 Storage 数据
4. 在数据网格中查看和搜索数据

## 优势特点

✅ 单文件部署 - 只需一个 exe 文件
✅ 自包含运行时 - 无需安装 .NET
✅ 便携性强 - 可以放在 U 盘中使用
✅ 完整功能 - 包含所有原始功能
✅ 启动速度快 - 优化的单文件打包

## 注意事项

- 建议在运行程序前关闭所有 Chrome 浏览器窗口
- 程序读取的数据可能包含敏感信息，请妥善处理
- 仅在合法和授权的环境中使用本工具
- 首次运行可能需要几秒钟解压内部文件

## 故障排除

如果遇到问题：
1. 确保以管理员权限运行
2. 检查 Chrome 是否已正确安装
3. 确认系统为 Windows 10/11 x64
4. 检查防病毒软件是否阻止了程序运行

## 技术信息

- 打包方式：.NET 6.0 单文件自包含发布
- 目标平台：Windows x64
- 包含组件：WPF 运行时、SQLite 支持、所有依赖库
- 文件大小：约 155MB（包含完整运行时）
